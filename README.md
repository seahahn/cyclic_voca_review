# Cyclic Voca Review

## 작품 소개

외국어 단어 학습 및 주기적인 복습을 목적으로 만들어진 앱입니다.

에빙하우스의 망각 곡선 가설에 따라 외국어 단어 학습을 효과적으로 했던 경험을 토대로 제작하였습니다.

---
